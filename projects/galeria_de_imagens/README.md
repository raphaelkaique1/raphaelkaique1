# Galeria de Imagens

Este é um projeto simples de uma galeria de imagens, desenvolvido com HTML, CSS e JS. A galeria exibe imagens em um formato de carrossel horizontal, com efeitos de hover para aumentar a imagem e aplicar sombras. A página também tem opções de exibição de temas claro e escuro.

## Funcionalidades
- Exibição de várias imagens em uma linha.
- Efeito de zoom nas imagens ao passar o mouse (hover).
- Seleção de tema claro ou escuro.
- Layout responsivo, ajustando-se ao tamanho da tela do dispositivo.
